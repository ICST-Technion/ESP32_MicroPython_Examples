This code demonstrates how to connect an ESP32 to a Wi-Fi network and
perform an HTTP GET request using MicroPython. The device connects to a
specified Wi-Fi network and sends a request to the
\"http://httpbin.org/get\" API, printing the response.

connect_wifi()

Purpose: Connect the ESP32 to a specified Wi-Fi network.

What it does: 1.Activates the Wi-Fi interface (STA_IF mode for client
connection). 2.Checks if the device is already connected. If not:
2.1.Attempts to connect using the provided SSID and password. 2.2.Waits
until the connection is established. 3.Once connected, prints the
device\'s IP address and network configuration (IP, subnet mask,
gateway, DNS).

send_http_get_request()

Purpose: Perform an HTTP GET request to a predefined server URL.

What it does: 1.Defines the target URL (http://httpbin.org/get) to send
the request. 2.Uses the urequests.get() method to send the GET request.
3.Prints the HTTP response\'s status code and body content.

send_http_get_request() Execution

1.Sends an HTTP GET request to \"http://httpbin.org/get\". 2.If the
request succeeds: Response Status Code: Prints 200, indicating success.
Response Content: Prints the server\'s response, usually in JSON format
containing details about the request (e.g., headers, IP address, etc.).
